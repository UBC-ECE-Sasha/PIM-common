|Directory        | Purpose                                             |
|-----------------|-----------------------------------------------------|
|common/include   | Macros and function definitions for common code     |
|dpu/include      | Macros and function definitions for DPU-only code   |
|host/include     | Macros and function definitions for host-only code  |
